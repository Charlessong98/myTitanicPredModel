# myTitanicPredModel
Analyze and modeling the Titanic survival data set.
## Package require
NumPy
Pandas
sklearn
Matplotlib
Seaborn
## Main method and results
Having feature engneering on the Titanic data set. Analyzed survival rate along with age, sex, title, family name, ticket, and other exsisting feature and add some new feature to make model fit better on this data. Used the Random forest model, and Linear regression model, find that the latter works better on this project. 

Finally, the result submitted to the Kaggle competition got grade 0.80622, which means 80.622% of time the model gives the right answer. Ranking 405 on the LeaderBoard.
